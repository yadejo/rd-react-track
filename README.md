# Realdolmen React Track - Exercise
## Pokemon REST API
### Setup
to start the API, execute following actions: 
- `$ cd ./api`
- `$ npm install`
- `$ nmp run start`

### Routes

| Route             | Description                           |
| -------------     |---------------------------------------| 
| `/pokemon`        | Get a list of all available Pokemon   |
| `/pokemon/{id}`   | Get details of a given Pokemon        |
